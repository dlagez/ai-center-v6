## 说明

LLM 统一接入：LiteLLM SDK
文档解析：Docling SDK
向量库：Qdrant
chunking：LangChain Text Splitters
索引构建：
    主检索：Qdrant hybrid retrieval
    补充检索：BM25Retriever
    后处理：Rerank / compression retriever
RAG：LangGraph
